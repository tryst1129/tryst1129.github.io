Useful links for windows

rufus (install)

Software that helps you burn image files to flash drives and the like. It can also help you install windows without a microsoft account

Ventoy (install)

Software you can use in conjunction with rufus to hold multiple bootable image files on the same flash drive

Microsoft Activation Scripts (install)

Utility that lets you activate windows and the office suite

winutil (install)

Utility that lets you uninstall edge, and do lots of other useful things

CrystalDiskInfo (install)

Software that helps you monitor your hard drive health. NVME drives are only partially supported but it works well for me

SpotX (install)

Utility that installs and modifies the spotify client

SSHFS-Win (install)

Software that enables mapping of SFTP servers in windows
