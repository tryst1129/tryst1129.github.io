Useful links for windows

rufus

This is a software that helps you burn image files to flash drives and the like. It can also help you install windows without a microsoft account.

Ventoy

This is a software you can use in conjunction with rufus to hold multiple bootable image files on the same flash drive.

Microsoft Activation Scripts

This is a utility that lets you activate windows and the office suite. Use it at your own risk.

winutil

This is a utility that lets you uninstall edge, and do lots of other useful things. Use it at your own risk.

CrystalDiskInfo

This is a software that helps you monitor your hard drive health. NVME drives are only partially supported but it works well for me.

SpotX

This is a utility that installs and modifies the spotify client. There's a seperate version for MacOS. Once again, use it at your own risk.
